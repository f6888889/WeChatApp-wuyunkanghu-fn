# 银舞沐心 - 设计规范

## 理念

我们不做养老App，我们做的是：**让爷爷奶奶重新爱上自己**。

界面要像午后阳光一样温暖，操作要像握手一样简单。

---

## 配色方案

### 主色调

| 用途 | 色值 | 场景 |
|------|------|------|
| 暖阳橙 | #E07A5F | 主要按钮、强调 |
| 米白底 | #FDF6EC | 背景 |
| 深褐字 | #3D2C2E | 正文 |
| 柔和灰 | #8B7D7B | 次要文字 |
| 草绿 | #81B29A | 健康数据、成功状态 |
| 珊瑚红 | #F2CC8F | 提醒、警示 |

### 渐变色（用于卡片背景）

```
background: linear-gradient(135deg, #FDF6EC 0%, #F8E8D6 50%, #F5DFC5 100%);
```

### 噪点纹理

所有背景添加 3% 噪点覆盖，增加温度感。

---

## 字体规范

- **标题**: 思源黑体 Bold / 38-48px
- **正文**: 思源黑体 Regular / 32-36px（比普通App大40%）
- **按钮文字**: 36px
- **行高**: 1.8（给眼睛留呼吸空间）

---

## 间距系统

- 页面边距: 40px
- 卡片间距: 32px
- 组件内边距: 32px
- 按钮高度: 96px（足够大的点击区域）
- 圆角: 24px（柔和感）

---

## 图标规范

图标库: Iconify
主图标集: `lucide` （线条柔和，适合老年人）
备用: `mdi` `tabler`

尺寸: 48-64px（比常规大2倍）

---

## 页面结构

### 1. 首页
- 问候语（带时间感知：早/午/晚安）
- 今日推荐舞蹈（1个大卡片）
- 快捷入口（4个不等分布）
- 今日健康提示

### 2. 搜索页
- 巨大语音按钮（占屏幕40%）
- 最近搜索（横向滚动）
- 热门推荐（不规则网格）

### 3. 舞蹈课程页
- 分类标签（横向滚动）
- 课程卡片（瀑布流，不等高）
- 难度提示（用水果大小比喻：苹果/橙子/西瓜）

### 4. AI健康伴侣页
- 数字人形象（居左，40%宽度）
- 健康数据卡片（右侧堆叠）
- 快捷提醒列表

### 5. 我的页面
- 用户头像+昵称（不对称布局）
- 学习进度环形图
- 功能入口（两列，错落）

---

## 动画规范

- 所有动画: 400-600ms（老年人需要时间感知）
- 缓动函数: `cubic-bezier(0.4, 0, 0.2, 1)` 或自定义「棉花糖」
- 页面切换: 从右滑入，300ms
- 卡片出现: fade + translateY(20px)，stagger 100ms
- 按钮点击: scale(0.96) + 轻微阴影变化

---

## 交互规范

- 点击区域: 最小 88px × 88px
- 操作反馈: 颜色变化 + 轻微缩放 + 震动
- 语音输入: 长按说话，松开发送
- 翻页: 支持滑动，禁用极速滑动

---

## 文案风格

每句话不超过15字，口语化，像跟爷爷奶奶说话：

| 场景 | 文案 |
|------|------|
| 首页问候 | "今天想动一动吗？" |
| 搜索提示 | "说出您哪里不舒服" |
| 课程推荐 | "这个动作对膝盖好" |
| 完成学习 | "太棒了！您学得真快" |
| 坚持打卡 | "连续坚持3天了，继续加油" |

---

## 组件定制

### 大按钮
- 高度96px
- 圆角24px
- 轻微阴影
- 按下态：scale(0.96)

### 语音输入球
- 直径: 屏幕宽度 × 0.6
- 脉冲动画（呼吸感）
- 录音时：波纹扩散

### 课程卡片
- 高度不统一（200-280px随机）
- 封面图占比60%
- 右下角：难度标签（水果图标）

---

## 技术要求

### 后端 Spring Boot
- JDK 17+
- Maven
- JSON文件存储
- CORS配置
- 端口: 8080

### 前端 微信小程序
- 原生开发（不用框架）
- 分辨率适配: rpx
- 图片: 使用Picsum占位

### API设计
- 前后端分离
- RESTful风格
- 统一响应格式
